Questions by type & variant

Each type can hold several variants; one variant is drawn per type per exam. Points are set per type — every variant of a type is worth the same. Each question is a LaTeX snippet (just the body — no \documentclass or \begin{document}). Available in the preamble are, among others, graphicx (\includegraphics), amsmath, listings (\lstinputlisting), tikz, and enumitem.

Assets (optional)

Files your questions reference by name — images for \includegraphics, or .txt for \lstinputlisting (e.g. R output). They are sent with the exam request and discarded afterwards.
